Personnaliser AbiWord

AbiWord est personnalisable selon vos besoins ou vos habitudes. 

Tout d'abord, il est possible de masquer ou d'afficher les règles horizontales et verticales. Pour cela, il suffit de décocher ou de cocher l'option "Règle" dans le menu Affichage. 

Il est possible également d'afficher ou de masquer les caractères invisibles, comme les espaces, les tabulations ou des sauts (de sections, de colonnes ou de pages). Pour cela, il suffit de décocher ou de cocher l'option "Afficher/Masquer ¶ " dans le menu Affichage.

Tous les autres paramètres sont personnalisables dans le menu Outils puis "Options..."

La fenêtre ci-dessous apparaît alors:

Cette fenêtre est constituée de quatre onglets:

* L'onglet Spelling (Correction automatique)
* L'onglet Preferences Schemes (Préférences)
* L'onglet View (Affichage)
* L'onglet Other (Autre)

En bas de cette fenêtre se trouve quatre boutons

* Le bouton "Save" permet de sauver les préférences et permet ainsi de les retrouver après un redémarrage d'AbiWord.
* Le bouton "Defaults" permet de rétablir les options par défaut (ou d'origine) d'AbiWord.
* Le bouton "OK" permet de valider les changements ; mais ces changements ne seront plus valides au prochain redémarrage d'AbiWord.
* Le bouton "Cancel" (Annuler) qui ne prend pas en compte les modifications.

  1. L'onglet Spelling (Correction automatique) :

Il est composé de plusieurs options à cocher ou décocher, parmi lesquelles :

* Check spelling as you type: active ou désactive la vérification automatique en cours de frappe.
* Ignore words in UPPERCASE: vérifie ou ne vérifie pas les mots en majuscules lors de la vérification en cours de frappe.
* Ignore words with numbers: vérifie ou ne vérifie par les mots contenant des nombres lors de la vérification en cours de frappe.

Les autres options à cocher apparaissent en grisé car elles n'ont pas été encore implémentées dans la dernière version de AbiWord. Il en est de même pour les boutons Dictionnary, Edit et Reset.

  1. L'onglet Preference Schemes (Préférences):

Cet onglet comporte l'option à cocher "Automatically save this Scheme" (Sauvegarder automatiquement le thème). Si cette option est cochée, la sauvegarde de l'environnement AbiWord est effective. Si par exemple vous choisissez dans les options de masquer la règle (dans le menu Affichage puis Règle...), la règle continuera d'être masquée au prochain redémarrage de AbiWord.

Le champ "Current Preferences Scheme" (Thème Préféré Courant) qui permettra de définir plusieurs thèmes de personnalisation de AbiWord apparaît en grisé car il n'a pas été encore implémenté dans la dernière version de AbiWord.

  1. L'onglet View (Affichage):

Cet onglet est lui aussi composé de plusieurs options à cocher ou décocher, réparties dans deux rubriques: la rubrique "Show/Hide..." (Cacher/Masquer...) et la rubrique "View..." (Affichage)

* Ruler (Règle): Affiche ou masque la règle, tout comme dans le menu Affichage puis "Règle".
* Standard Toolbar (Barre d'outils Standard): Affiche ou masque la barre d'outil standard, tout comme dans le menu Affichage, Barre d'outils puis "Standard".
* Format Toolbar (Barre d'outils Mise en forme): Affiche ou masque la barre d'outil mise en forme , tout comme dans le menu Affichage, Barre d'outils puis "Mise en forme".
* Extra Toolbar (Barre d'outils extra): Affiche ou masque la barre d'outil extra, tout comme dans le menu Affichage, Barre d'outils puis "Extra".
* Status bar (Barre d'état): Affiche ou masque la barre d'état, tout comme dans le menu Affichage puis "Barre d'état".
* Cursor blink (Curseur clignotant): Active ou désactive le curseur clignotant.
* Unprintable characters (caractères non imprimable):  Affiche ou masque les caractères invisibles (comme les espaces, les tabulations ou les différents sauts). Tout comme  dans le menu Affichage puis "Afficher/Masquer ¶ ". (Pas encore implémenté dans la dernière version de AbiWord.)
* Invisible Layout Marks (limites de texte):  Affiche ou masque les limites de texte.

Le champ "units" permet de choisir l'unité de mesure de la règle. Pour cela cliquer sur la petite flèche à droite du champ "units" puis sélectionner l'unité de mesure désirée parmi le inch, le centimètre, le pixel ou le pico.

Les autres options à cocher apparaissent en grisé car elles n'ont pas été encore implémentées dans la dernière version de AbiWord.

  1. L'onglet Other (Autre):

Cet onglet comporte l'option à cocher "Enable smart quotes" (Active l'Astuce du jour). Si cette option est cochée, une astuce du jour sera affichée lors du démarrage d'AbiWord. (Pas encore implémentée dans la dernière version de AbiWord.)

 

D'autres éléments, comme changer la langue d'AbiWord ou faire fonctionner la vérification automatique dans une autre langue , sont personnalisables. Contrairement aux éléments ci dessus personnalisables par l'intermédiaire d'une boite de dialogue, ces éléments peuvent être personnalisés en éditant un fichier à l'aide d'un éditeur de texte.
Des informations relatives à ce type d'intervention sont disponibles dans la section Foire Aux Questions (FAQ) de ce même manuel.
