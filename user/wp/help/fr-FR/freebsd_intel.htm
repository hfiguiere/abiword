Installation d'AbiWord sur FreeBSD / Intel

Bienvenue dans la communauté des utilisateurs d'AbiWord!

La dernière version d'AbiWord doit correctement s'installer et fonctionner sur les systèmes FreeBSD 3.2 avec processeur Intel compatible. Abiword a été "packagé" dans plusieurs formats de distribution afin de simplifier son installation sur votre système.

  • TGZ -- Package FreeBSD 3.2 natif
  • TAR -- Tourne sur les anciennes versions de FreeBSD en liens-statiques

Une version plus exhaustive du processus d'installation est disponible dans le fichier README.

TGZ

Le processus d'installation TGZ est très simple.

  1. Télécharger l'archive (AbiSuite-x.x.x.tgz).
  2. En tant que superviseur, taper la commande: 
      pkg_add AbiSuite-x.x.x.tgz
  3. Cela installera AbiWord et ses fichiers dans le dossier /usr/local/AbiSuite/. Des liens par défauts appelés AbiWord et abiword seront stockés dans /usr/local/bin/ et pointeront sur les exécutables.
  4. Pour des informations complémentaires sur les pachages TGZ, veuillez consulter le fichier README.

TAR

Il existe deux versions des archives Tar: une version liens-statiques et une versions liens-dynamiques. La version dynamiques est moins grosse (2.7Mb contre 3.3Mb), mais utilise un nombre considérable de librairies partagées de votre système. Si vous utilisez FreeBSD 3.2 ou ultérieur, le package dynamique devrais fonctionner. Si vous rencontrez des problèmes d'exécutions avec le package dynamique, essayez le package statique.

Le processus d'installation de l'archive statique Tar parait complexe mais est actuellement assez simple.

  1. Télécharger l'archive (abisuite-x.x.x-FreeBSD_i386_static.tar.gz).
  2. Copiez l'archive dans un dossier accessible par vous et ouvrez l'archive: 
      gunzip -c abisuite-x.x.x-FreeBSD_i386_static.tar.gz | tar xvf -

    ... ou, si votre commande tar accepte les tags 'z':

      tar xvzf abisuite-x.x.x-FreeBSD_i386_static.tar.gz

    Les deux commandes créeront un dossier abisuite-x.x.x-FreeBSD_i386_static.

  3. Allez dans ce dossier en tant que superviseur.
  4. Lancez le script d'installation: 
      ./install.sh
  5. Par défaut, Cela installera AbiWord dans le dossier /usr/local/AbiSuite de votre disque dur. Vous pouvez changer le dossier par défaut si souhaité.
  6. Vous pourrez également créer les liens vers les exécutables dans un dossier bin . Par défaut ce lien est /usr/local/bin.
  7. Merci de consulter le fichier README pour des informations complémentaires.

Le processus d'installation de l'archive dynamique (abisuite-x.x.x-FreeBSD_i386_dynamic.tar.gz) est identique. Remplacer simplement _dynamic for _static dans les étapes 1 et 2 ci-dessus. Si vous voulez plus d'informations à propos des librairies et des versions que la version dynamique utilise, veuillez consulter le fichier README.
