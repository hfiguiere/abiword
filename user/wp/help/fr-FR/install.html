Installer AbiWord

Le traitement de texte Open Source et Multi-plateformes

AbiWord est totalement gratuit.

AbiWord étant disponible sur plusieurs plate-formes, voyons en détail, son installation sur chacune de ces plate-formes:

Windows

  • Windows 95, 98, NT

Linux

  • Linux / Intel
  • Linux / PowerPC
  • Linux / Alpha

BeOS

  • BeOS / Intel (experimental) 
  • BeOS / PPC

UNIX

  • Solaris
  • FreeBSD / Intel
  • Other
  • NetBSD

PS:

Les sections suivantes décrivent comment télécharger et installer AbiWord sur votre machine. Pour la plupart des personnes, c'est tout ce dont vous aurez jamais besoin.

Comme AbiWord est en Open Source, vous avez toujours la possibilité de télécharger son code source, de le modifier autant que vous le voulez et ensuite de compiler votre propre et unique version personnelle de AbiWord. Pour un complément d'information sur la personnalisation de AbiWord, veuillez consulter la section des développeurs sur le site AbiSource.
