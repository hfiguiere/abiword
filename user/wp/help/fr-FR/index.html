Description d'AbiWord

AbiWord est un petit traitement de texte (jusqu'à présent), mais rapide et complet. Il fait partie du projet AbiSource qui est lui même sponsorisé par la société SourceGear. Le but de ce projet est le développement d'une suite bureautique Open Source et multi plate-formes. Ce projet commence avec AbiWord, le traitement de texte. Parmi ses caractéristiques, on peut citer son interface "look & feel" (voyez la capture d'écran), ses possibilités de mise en forme de pages et de paragraphes, un correcteur orthographique, une règle interactive, l'intégration de styles, sa capacité à Annuler / Répéter illimitée, une fonction Rechercher / Remplacer et l'insertion d'images. Il est également capable d'importer des documents Microsoft Word (doc) et RTF (Rich Text Format) et de sauvegarder des documents Internet HTML. 

Mais la différence la plus significative d'AbiWord par rapport à presque tous les autres traitements de texte disponibles est son format de fichiers natifs. Un document sauvegardé sous AbiWord a une extension par défaut *.abw écrit en XML et donc en format ASCII. Ces fichiers peuvent alors être lus par n'importe quel éditeur de texte. Cela ne signifie pour autant pas que AbiWord est un éditeur XML. AbiWord est un traitement de texte qui utilise le XML comme une syntaxe pratique de représentation de documents. C'est un virement total par rapport aux autres traitements de texte traditionnels et cela assure que lorsqu'un document est écrit avec AbiWord, il n'est assujetti à aucun traitement de texte particulier ;  traitement de texte qui deviendra obsolète dans les cinq années à venir.

Enfin, AbiWord est totalement gratuit et multi plate-formes: il fonctionne sous Windows et la plupart des systèmes Unix.

  • BeOS / Intel (expérimental)
  • BeOS / PPC (expérimental)
  • Linux / Alpha (Temporairement indisponible)
  • Linux / Intel
  • FreeBSD / Intel
  • Solaris
  • Windows 95, 98, NT
  • Autre

Les prochaines années verront l'informatique personnelle muter profondément avec le développement et le déploiement de plusieurs systèmes d'exploitation (OS). AbiWord sera donc capable de tourner sur ces principaux OS. Encore une fois, le choix des fichiers natifs XML assurera une portabilité de ces fichiers sur ces différentes plate-formes alors que  les traitements de texte mono plate-forme traditionnels resteront cloisonnés.

Le concept Open Source

Les logiciels faisant partie du concept Open Source se distinguent des traditionnels autres logiciels commerciaux du fait que leurs codes sources (les lignes d'instructions qui définissent le logiciel en lui-même) sont librement disponibles de même que les logiciels eux-mêmes. Pour la plupart des utilisateurs, cela importe peu ; toutefois, tout le monde bénéficie des améliorations incessantes que produit chaque volontaire sur les logiciels d'origine. Tout le monde peut participer à l'amélioration du produit. Cette notion s'oppose à celle de logiciel propriétaire, protégé par une licence interdisant l'accès à son code source, les ajouts ou les retraits, la copie et la redistribution. Pour plus d'informations sur le concept Open Source, visitez le site www.opensource.org

AbiWord est donc un logiciel Open Source. Ce n'est pas un shareware: vous n'avez aucune obligation légale, morale ou de toutes autres sortes. Il est totalement gratuit. Nous voulons simplement que tout le monde puisse utiliser ce logiciel: vous pouvez même le copier pour d'autres personnes susceptibles d'être intéressées.

AbiWord est en train de naître d'un modèle de développement que beaucoup de personnes méconnaissent. Le résultat des contributions de nombreuses personnes de par le monde est que des  mises à jour fréquentes apparaissent. Les utilisateurs d'AbiWord devront surveiller sans cesse qu'une nouvelle version soit sortie, et devront également consulter le site d'AbiSource fréquemment pour être sûrs qu'ils utilisent la version la plus récente d'AbiWord.
