Les colonnes avec AbiWord

Il est possible de disposer un long document en colonnes pour améliorer sa lisibilité. AbiWord est capable de gérer jusqu'à trois colonnes sur une même page. La répartition du texte entre les différentes colonnes est gérée automatiquement par AbiWord : si une colonne est complète, le texte se poursuit en haut de la colonne suivante.

Par défaut, le mode "une colonne" est activé lors du lancement d'AbiWord. Pour définir des colonnes, il suffit de se positionner dans la section à transformer ou de sélectionner le texte à disposer en colonne puis de cliquer sur les icones "Colonnes" de la barre d'outils Standard". 

  1. L'icone pour définir une colonne
  2. L'icone pour définir deux colonnes
  3. L'icone pour définir trois colonnes

Pour augmenter ou réduire l'espace entre deux colonnes, cliquer sur la zone "marge de colonnes" dans la règle puis faire glisser.
