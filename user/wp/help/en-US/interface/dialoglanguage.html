Language Dialog

This dialog lets you tell AbiWord the language in which you are writing. This changes the dictionary used for spell-checking, and other properties which depend on the language.

Select Language

This menu lists all the languages that AbiWord knows about. Click on one to select it. If your language is not listed here, then your version of AbiWord does not know about your language. Check the AbiWord website to see if support has been added since your version was produced.

If you select 'no proofing', AbiWord will not try to spell check your document. If your language is not available, this is the best option. 'no proofing' is the last item in the menu.

OK

This button closes the Language dialog and changes the current language.

Cancel

This button closes the dialog without changing the current language.
