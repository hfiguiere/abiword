Activate Window Dialog

This dialog allows you to choose a window from the list of all those open.

Activate:

This is a list of all the windows that are currently open in AbiWord. Click on one to select it.

OK

Press this button to make the selected window active. This brings it to the front, and places the insertion point within it.

Cancel

Close the dialog without changing the active window.
