Web Menu

The web menu provides two commands which allow you to easily turn your AbiWord documents into web pages.

Save as web...

This command prompts you to save your document under a new name, with the file type XHTML. AbiWord's built-in exporter automatically converts your file to XHTML when it is saved.

Preview web page...

This command opens the current document in your default web browser. You can thus see what it will look like as a web page. This command does not save an XHTML version of your document; you must use the Save as web... or Save As... command for that.
