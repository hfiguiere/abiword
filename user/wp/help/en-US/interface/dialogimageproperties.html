Image Properties Dialog

Height/Width:

These boxes show the current dimensions of the image. You can alter them by clicking on the arrows to the right of the boxes.

Preserve Aspect Ratio

If this option is checked, AbiWord will keep the relative sizes of the height and width constant. Thus, if you decrease the height, the width will also be decreased, so that the shape of the image remains the same.

OK

Press this button to close the dialog and apply your changes.

Cancel

Press this button to close the dialog without making any changes.
