Insert Symbol

This dialog allows you to insert characters into your document. This includes characters that you may not be able to type easily.

Font Menu

This drop-down menu lists the fonts from which you can choose the characters you insert. The selected font is displayed in the box.

Symbol Table

The large table shows all the symbols available in the selected font. Click on a symbol to select it for insertion. Double clicking on the symbol inserts it into the document, but does not close the dialog.

Preview Box

When a symbol is selected, a large version of that symbol is displayed in this box, so that you can see what you will insert.

Insert

Click this button to insert the currently-selected symbol.

Close

Click this button to close the dialog. Any symbols you have inserted remain in the document.
