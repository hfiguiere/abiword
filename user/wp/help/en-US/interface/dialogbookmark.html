Insert Bookmark

A bookmark marks a point or section of your document so that AbiWord can find it quickly. Bookmarks are normally invisible, although they are represented if you have set AbiWord to show formatting marks. In the Go To dialog, you can choose to go to a bookmark. Since bookmarks can be given informative names, this may be easier than trying to navigate by going to page numbers. Bookmarks can also serve as destinations for hyperlinks within a document.

To insert a bookmark, type a name for the bookmark in the box in the dialog. The bookmark will mark the selected text. If no text is selected, the bookmark will mark the location of the insertion point. Bookmark names should not contain periods (.), because any hyperlink name with a period is treated as a link to another file.

OK

Insert the bookmark and close the dialog.

Delete

Press this button to remove the selected bookmark from your document.

Cancel

Close the dialog without inserting a bookmark.
