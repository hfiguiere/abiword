Tabs Dialog

This dialog allows you to set the position and type of the tab stops in your document.

Tab stop position:

Type the position of the tab stop you want in the box below this heading.

Default tab stops:

This box shows the interval between the automatically generated tab stops. If you have no tabs set, tabbing in your document will move you this distance. These tabs remain to the right of a deliberately inserted tab stop, but are suppressed to its left.

Tab Stop list

This window to the left of the dialog is not titled on the dialog. It lists the positions of all set tab stops. If a created tab stop is selected, the settings of the Alignment and Leader sections change to show the settings for that tab.

Tab stops to be cleared:

This takes up screen space and does nothing.

Alignment

These radio buttons allow you to set the kind of tab stop.

Left alignment means that text starts at the tab stop and extends to the right.

Center alignment means that text is centered on the tab stop.

Right alignment means that text finishes at the tab stop and extends to the left.

Decimal means that the decimal point in whatever you type will be under the tab stop. This is useful for formatting financial information.

Bar means that AbiWord will draw a bar at the tab position, and insert the text after it as if there was a left tab just to its right. These tab stops can be used with topline and bottomline formatting to draw simple tables.

Leader

These buttons allow you to set the characters which are inserted between the point where you insert the tab and the beginning of the tabbed text.

Set

This button sets the tab stop you have selected. A tab is not set until it appears in the list of set tab stops. If you are modifying a tab, you must press this button to make the changes.

Clear

This button removes the selected tab stop.

Clear All

This button removes all of the user-defined tab stops.

Apply

This button applies the currently listed tab stops to the document, but does not close the dialog. You cannot type in the document while the tabs dialog is open, but any tabbed text will change to match the new tabs.

OK

This button accepts your changes and closes the dialog.

Cancel

This button closes the dialog without saving any changes to the tab stops that you have made since you last clicked 'Apply'.
