Change Case Dialog

The Change Case dialog affects the selected text, if any. If there is no text selected, it affects the word containing the insertion point. The insertion point counts as being in a word if it immediately next to the first or last letter, as well as if it is between two letters of the word.

Case Options

You can change the case of the selection in five different ways.

  • Sentence case. This option sets the first letter of the selection to upper case, and leaves the rest unchanged. It does not change the first letter of every sentence, it does change the first character of every paragraph whether or not it starts a sentence, and it does not change a sentence written entirely in upper case to one with an upper case first letter and lower case for the rest.
  • lowercase. This option changes all the letters in the selection to lower case.
  • UPPERCASE. This option changes all the letters in the selection to upper case.
  • Camel Caps. This option changes each word so that the first letter is upper case and subsequent letters are lower case. It works on words that are all in upper case.
  • tOGGLE cASE. This option changes lower case letters to upper case, and vice versa.

OK

Click this button to apply the case option you have selected, and close the window.

Cancel

Click this button to close the window without altering the selected text.
