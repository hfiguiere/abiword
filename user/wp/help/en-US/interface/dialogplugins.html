AbiWord Plugin Manager

This dialog controls the use of plugins in AbiWord. Before you can use a plugin, it must be installed on your computer. There are many plugins available for AbiWord, many of which are linked from the AbiWord website. The instructions given here assume that the plugins are available on your machine.

Any plugins that have been installed in the default locations (most installers will do this) are loaded when you launch AbiWord.

The dialog has two tabs, Plugin List and Plugin Details.

Plugin List

This tab allows you to activate and deactivate plugins. A plugin can only be used by AbiWord while it is active. Similarly, a plugin only uses system resources while it is active.

Active Plugins

This window lists all currently active plugins. Click on a plugin to select it.

Deactivate plugin

This button deactivates the selected plugin.

Deactivate all plugins

This button deactivates all currently active plugins, whether they are selected or not.

Install new plugin

This button opens a standard Open File dialog. You must find the plugins and open them in order to activate them. The dialog automatically opens in the default plugin directory for AbiWord. Most plugins will be automatically installed in this directory. If you have installed them elsewhere, you must change the directory yourself.

Note that, on Windows, the operating system may be set up to hide all .dll files from program dialogs. As AbiWord plugins are .dll files, this means that you will not be able to see the plugins in the directory.

Plugin Details

This window provides information about the plugin that is currently selected on the Plugin List tab.

Close

Clicking this button closes the dialog.
