Zoom Dialog

This dialog lets you control the magnification of your document on the screen.

Zoom to

These options fix the magnification. 200%, 100%, and 75% are relative to the printing size. You can also pick any percentage using the Percent: option at the bottom of the list. Click on the button, and then click on the up and down arrows to alter the value. You can also select the box and type a number.

'Page width' sets the magnification of the document so that the whole width of the page just fills the window. 'Whole Page' fits the entire page into the window, so that you can see the whole layout at once.

Preview

This window shows you how big the font named on the frame will appear at the zoom value you have selected. This only works for the percentage values, not for the zooms based on page size.

OK

This button closes the Zoom dialog, and changes the Zoom of the document to whatever you have selected.

Cancel

This button closes the Zoom dialog without changing the Zoom of the document.
