Columns Dialog

This dialog allows you to create newspaper-style columns in your document. The text will fill one column before moving on to the next. The column setting you select applies to the whole of the current section. If you want to change the number of columns in a document, you must insert a section break: see the Insert Break dialog.

Number of Columns

Press the button corresponding to the number of columns you want to select it.

Preview

This window shows you what a page will look like with the number of columns you have selected.

Line Between

If this option is selected, AbiWord draws narrow vertical lines between the columns of text.

Number of Columns

You can set the number of columns in this box, as well as by pressing the buttons. It is possible to have more than three columns if you set them here. You can type in the box or use the arrows to increase or decrease the number.

Space after Column

Set the space that AbiWord leaves between the end of a column and the following text. You can type in the box or use the arrow buttons to increase and decrease the setting.

Max Column size

Set the maximum height of a single column. If the text will not fit in this space, AbiWord starts a new column. This may mean moving to the next column across the page, or starting a new column below the one that is full. If a new column is started below the current column, the space specified in that option will be left.

If this is set to 0.0, there is no maximum height. Columns fill the whole page before starting a new column.

OK

Click this button to close the dialog and change the number of columns.

Cancel

Click this button to close the dialog without changing the number of columns.
