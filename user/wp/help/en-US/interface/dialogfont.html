Font Dialog

This dialog lets you change the appearance of the characters in your document.

Font

The commands on this tab change the appearance of the characters themselves.

Font:

This window lists all the fonts installed on your system and available to AbiWord. Click on one to select it.

Style:

This determines the overall look of the characters. It is not related to the styles which you can set from the Style dialog. Click on one to select it.

Regular: The text has its normal weight and shape.

Italic: The text slopes to the right.

Bold: The text is heavier and darker.

Bold Italic: A combination of bold and italic.

Size:

This is a list of the available sizes for a font. These are known as point sizes, and the larger the number, the larger the font. Click on one to select it.

Effects:

These are special effects which can be added to the characters. Click on the check box beside each one to turn it on or off. You may select as many of these as you wish.

Strikeout: A line through the characters, as if they are crossed out.

Underline: A line under the characters.

Topline: A line at the top of the line of text, higher than an overline.

Bottomline: A line at the bottom of a line of text, lower than an underline.

Overline: A line over the characters.

Color (Windows only)

This menu allows you to select the text color.

Script (Windows only)

This allows you to select the script that the font represents.

Text Color (Linux only)

This tab allows you to select the color of the characters. You can select it by clicking on the color wheel on the left, or by adjusting the sliders on the right. The right-hand half of the rectangle under the color wheel shows you what you have chosen.

Highlight Color (Linux Only)

This tab allows you to select a color that appears behind the characters. Unlike a background color, it only appears behind characters that have been typed, rather than covering the whole page. You select it in the same way as you choose the text color.

Sample

This window shows you some text. The formatting of this text is changed to show you what you have selected.

OK

This button closes the window and makes the changes to any selected text. If no text is selected, the changes affect anything you type from that point on.

Cancel

This button closes the dialog without making any changes.
