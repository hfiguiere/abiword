Window Menu

The Window menu gives you access to the various windows that AbiWord has open on your system.

New Window

This command creates a new window containing the current document. The name of the document is given in the title bar, followed by a colon and a number indicating the order in which the windows were created.

Any changes that you make to the document in one window are reflected in the others.

Window List

Below the New Window command is a list of all the windows currently open in AbiWord. Select a window to bring it to the front and put the insertion point within it.

If you have ten or more windows open, only the first nine opened are displayed on the menu. The tenth option becomes More Windows..., which opens the Activate Window dialog, so that you can select any window.
