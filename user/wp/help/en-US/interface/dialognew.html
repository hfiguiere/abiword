New File Dialog

The New File (or Choose A Template) dialog allows you to create files based on templates. A template sets the styles and initial text for a file. If you often write letters, for example, you might create a template with your address already entered.

Create new document from a Template

If you select this option, you must also select a template from the list below it. This includes all the templates you have defined, and any that are installed on your system. A few templates are installed with AbiWord. The selected template is the one from which the new document is created.

Open an existing document

This creates a new document with the same content and styles as the existing document you select. It has the same effect as the Import command.

Start with an empty document

This creates a new document based on the default template. This is the normal.awt template in your home directory, if there is one, or the normal.awt template installed with AbiWord if there isn't. If neither of these files exists, AbiWord defaults to built-in values.

OK

Pressing this button closes the dialog and creates a new document according to the options chosen.

Cancel

Pressing this button closes the dialog without creating a new document.
