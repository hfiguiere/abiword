Page Numbers Dialog

This dialog inserts page numbers on every page of your document.

Position

The drop-down menu here allows you to choose whether the page numbers appear in the header of the document, or the footer.

Alignment

The drop down menu for Alignment determines whether the page numbers are placed at the left,  right, or center of the header or footer.

Preview

This display shows the selected position of the page number relative to the main text of the page.

OK

This button inserts the page numbers and closes the dialog.

Cancel

This button closes the dialog without inserting the page numbers.
