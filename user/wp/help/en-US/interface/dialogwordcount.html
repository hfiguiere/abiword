Word Count Dialog

When the word count dialog is opened, it displays the statistics for your document at that point. If it is left open, the statistics can be updated, as described below.

Auto Update

If this option is selected, AbiWord keeps track of the number of words in your document as you type, and updates the statistics accordingly.

Seconds between updates

This box tells you how often AbiWord will update the number of words if you have selected Auto Update. You may type a number, or use the arrows beside the box to increase or decrease the interval by tenths of a second.

Statistics

The document name and path is displayed at the top of the statistics box.

'Words' gives the number of words in the document, where a word is defined as a string of letters or numbers between two spaces or punctuation marks. Thus, "1" is a single word, but "mis-spelled" is two, and "13/4/95" is three.

'Paragraphs' gives the number of paragraphs, as defined by the number of returns you have typed.

The characters entries are exactly what they sound like. Only printable characters are counted.

'Lines' is the number of lines of text in the document. A blank line inserted by typing return counts; blank space inserted between text, such as if the text is double-spaced, is not.

'Pages' is the number of pages that the document will occupy when printed.

Update

This button is only available if Auto Update is not selected. Pressing the button updates the word count for your document.

Close

This button closes the word count dialog.
