Personnalising AbiWord

AbiWord can be personnalised according to to your needs or your practices. Firstly, it is possible to hide or display the horizontal and vertical rulers. To do so, check the option " Ruler " under the "View" menu.

All the other other features that are able to be customised, are found under " Options... " in the "Tools" menu.

If you click on this the window below then appears:

This window contains three options:

* Automatic spell checker
* Preference Schemes
* View

In bottom of this window there are four buttons

* The " Save " button makes it possible to save the preferences so they will stay set even after you restart of AbiWord.
* The " Defaults " button makes it possible to restore the default options of AbiWord.
* The " OK " Button makes it possible to validate the changes; but these changes will not be saved, so you will have to reset them when you next restart Abiword.
* The cancel button does not implement any modifications.

  1. Automatic Spell Checker :

There are several options which can be checked or unchecked:

* Check spelling as you type: automatic spell checking
* Ignore words in UPPERCASE: does not spell check words in uppercase
* Ignore words with numbers: does not spell check words containing numbers

The other options are grayed out as they have not yet been implemented in AbiWord. This applies to the Dictionnary, Edit and Reset options.

  1. Preference Schemes:

This sheet contains the option " Automatically save this Scheme ". If this option is checked, then saving the AbiWord preferences will work. If for example you choose in the options to hide the ruler, then ruler will remain hidden when you next start AbiWord.

The " Current Scheme Preferences " field makes it possible to define several types of personalisation. This appears grayed in because it has not been implmented yet.

  1. View:

This sheet is also composed of several options that can be checked or unchecked, under two headings: the heading " Show/Hide... " (Cacher/Masquer...) and the heading " View... "

* Ruler: displays or hides the ruler
* Cursor blink : makes the cursor blink or stay constant.

The "units" field makes it possible to choose which measuring unit is used for the rule. Click on the small arrow on the right of the "units" field to select the measuring unit. You can choose between inches, centimetres, pixels or picos.

The other options are grayed out as they are not yet implemented in AbiWord.
