Styles

Styles serve to automate paragraph formatting. By applying styles, you can format your documents faster and also make the appearance of your documents more consistent.

AbiWord currently has six predefined styles:

  1. The Normal style:

    This is the basic style which you use for all your paragraphs. By default, the whole document is this style. 
    Characteristics: Font Times New Roman, Size 12.

     

  2. The Block Text style:

    It can be used, for example, when inserting quotations in a text. 
    Characteristics: Font Times New Roman, Size 12 and indents 1 inch on left and right.

     

  3. The Plain Text style:

    This is an alternative to the default Normal style; it defines another paragraph style.
    Characteristics: Font Courier New, Size 12.

     

  4. The Heading1 style:

    It can be used for section titles.
    Characteristics: Font Arial, Size 16, Bold.

     

  5. The Heading2 style:

    This can be used for subsection titles.
    Characteristics: Font Arial, Size 16, Bold, Italic.

     

  6. The Heading3 style:

    This style can be used for paragraph titles.
    Characteristics: Font Arial, Size 14.

 

To apply a style to a paragraph, place the point of insertion in the paragraph, then choose the style by clicking on the little arrow to the right of the Style command on the Formatting toolbar, then select the desired style from the list of choices.
