Installation von AbiWord auf BeOS / PPC

Die aktuelle Version von (0.7.9) sollte problemlos auf jedem BeOS R4 System mit einem PPC Prozessor installieren und funktionieren.

Bitte beachten Sie, daß diese Version von AbiWord nur in einer ZIP Datei verfügbar ist: Zukünftige Versionen werden in einem   installations-freundlicherem Format erhältlich sein.

Die Installation der ZIP Datei ist sehr einfach:

  1. Herunterladen der Datei ( AbiSuite-0.7.9-ppc.zip , 0.76MB).
  2. Unzippen der Datei in /boot/apps. Dies wird das Verzeichnis AbiSuite anlegen.
  3. Um AbiWord zu starten, /boot/apps/AbiSuite/bin/AbiWord eintippen.
  4. Für weitere Hinweise siehe README .
