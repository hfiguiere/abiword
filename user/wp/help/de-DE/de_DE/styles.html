Dokumentenstil

Um Dokumente schneller zu formatieren und ihr Aussehen konsistenter zu machen, haben Sie die Möglichkeit, einen Dokumentenstil zu definieren. Dieses kann wie bei anderen Formatierungen getan werden, bevor Sid mit dem Schreiben des Dokumentes beginnen oder aber auch nachdem das Dokument bereits fertig geschrieben ist. Dieser Stil wird automatisch Paragraphen formatieren.

  • Bevor Sie mit dem Schreiben beginnen

    Wählen Sie den Stil aus indem Sie auf den Pfeil rechts vom 'Style'-Befehl auf der Formatierungs Werkzeugleiste. Wählen Sie den gewünschten Stil aus der Liste aus. Die Formatierung beginnt am Anfang dieses Paragraphen.

  • Nachdem das Dokument bereits geschrieben ist

    Plazieren Sie den Cursor in den Paragraphen, den Sie formatieren wollen. Wählen Sie den Stil aus indem Sie auf den Pfeil rechts vom 'Style'-Befehl auf der Formatierungs Werkzeugleiste. Wählen Sie den gewünschten Stil aus der Liste aus. Nur der Paragraph, in dem sich der Cursor befindet, wird formatiert.



Vordefinierte Stile

  1. Normal

    Dies ist der normale Stil, der für alle Paragraphen benutzt wird. Standardmäßig besitzt das gesamte Dokument diesen Stil.
    Characteristics: Font-Times New Roman, Size-12.

     

  2. Block Text

    Dies kann verwendet werden um zum Beispiel ein Zitat einzufügen.
    Characteristics: Font-Times New Roman, Size-12, 1 Zoll links und rechts eingerückt.

     

  3. Plain Text

    Dies ist eine Alternative zum Normalen Stil; Es definiert einen anderen Paragraphen Stil.
    Characteristics: Font-Courier New, Size-12.

     

  4. Heading1

    Dies kann für Überschriten verwendet werden.
    Characteristics: Font-Arial, Size-16, fett.

     

  5. Heading2

    Dies kann für Unter-Überschriften verwendet werden.
    Characteristics: Font-Arial, Size-16, fett, kursiv.

     

  6. Heading3

    Dies kann für Unter-Unter-Überschriften verwendet werden.
    Characteristics: Font-Arial, Size-14.
