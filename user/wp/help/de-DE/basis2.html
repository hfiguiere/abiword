Editierbefehle

* Rückgängig
* Erneut durchführen
* Ausschneiden
* Kopieren
* Einfügen


Rückgängig

Es gibt 3 Wege, tippen, editieren oder formatieren rückgängig zu machen:
  1. Im 'Edit' Menu   "Undo" anklicken
  2. Das Icon "Undo"  anklicken
  3. Das Tastaturkürzel "Ctrl" und "Z" verwenden


Erneut durchführen

Es gibt 3 Wege, Effekte des "Rückgängig"-Befehls wiederherzustellen:
  1. Im 'Edit' Menu   "Redo" anklicken
  2. Das Icon "Redo"  anklicken
  3. Das Tastaturkürzel "Ctrl" und "Y" verwenden


Ausschneiden

Um ein Word, Paragraphen oder anderes Objekt (z.B. ein Bild) auszuschneiden, wählen Sie das Objekt aus und führen eine der drei Aktionen durch:
  1. Im 'Edit' Menu   "Cut" anklicken
  2. Das Icon "Cut"  anklicken
  3. Das Tastaturkürzel " Ctrl " und "X" verwenden oder in Windows, SHIFT und Delete

Dies wird das Objekt löschen. Es ist jedoch in einem Zwischenspeicher und kann irgendwo eingefügt werden mit dem Einfügen Befehl. command.



Kopieren

Der Kopierbefehl wird zum kopieren eines Objektes verwendet. Dieser Befehl muß mit dem Einfügen Befehl kombiniert werden. Um ein Wort, Paragraphen oder ein sonstiges Objekt (z.B. ein Bild) zu kopieren, wählen Sie dieses Objekt aus und führen eine der drei folgenden Aktionen durch:
  1. Im 'Edit' Menu   "Copy" anklicken
  2. Das Icon "Copy" anklicken
  3. Das Tastaturkürzel "Ctrl" und "C" verwenden oder in Windows Ctrl und Ins.

Das Objekt wird nicht verschwinden und dennoch im Zwischenspeicher des Computer abgelegt. Sie können es an beliebiger Stelle wieder einfügen.



Einfügen

Ist ein Objekt im Zwischenspeicher des Computer gespeichert (indem Sie entweder den Ausschneiden oder Kopieren Befehl verwendet haben), so können Sie das Objekt an der gewünschten Stelle wieder einfügen. Hierfür positionieren Sie den Cursor an der Stelle, an der Sie da Objekt haben wollen und wählen eine der folgenden Aktionen:
  1. Im 'Edit' Menu "Paste" anklicken
  2. Das Icon "Paste" anklicken
  3. Das Tastaturkürzel "Ctrl " und "V" verwenden oder in Windows Shift und Ins.
Dieser Befehl ist nur aktiv, wenn sich in der Zwischenablage ein Objekt mit dem "Ausschneiden" auder "Kopieren" Befehl befindett. Das letzte Objekt, welches ausgeschnistten oder kopiert wurde, wird eingefügt.
