Das 'Single Document Interface'

AbiWord ist ein 'Single Document Interface (SDI)'. Das bedeutet, daß ein AbiWord Fenster gleichzeitig nur ein einziges Dokument beinhalten kann. Wenn ein anderes Dokument geöffnet wird, so erscheint ein neues AbiWord Fenster. Umgekehrt wird beim Schließen des Fensters auch nur dieses Dokument geschlossen.

Das ist das Gegenteil eines Multiple Document Interface (MDI) Programs wie Microsoft Word 97, bei dem alle Dokumente in ein und demselben Fenster bearbeitet werden. Das Schließen des Dokumentes beendet nicht Microsoft Word.

Die SDI Technologie wurde für AbiWord aus zwei Hauptgründen gewählt::

  1. Da AbiWord für verschiedene Plattformen entwickelt wird (Windows, UNIX und bald auch Mac OS), wurde das Hauptaugenmerk auf die Einheitlichkeit der Versionen gelegt. Da UNIX auf dem single document interface (SDI) basiert, wurde dieses gewählt. Jedoch besitzt jedes geöffnete Dokument in AbiWord einen Knopf auf dem Taskbar.

  2. Die am meisten bekannte Anwendung heutiger Benutzer ist ohne Zweifel der Interner Browser. Die meisten heutigen Browser benutzen das SDI Konzept. Daher sind die meisten Neutzer dieses Konzept und immer weniger das MDI Konzept gewohnt. Dieses bestätigt die Wahl dieser Technologie. Neue Benutzer müssen sich jedoch keine Gedanken machen, wie AbiWord nun funktioniert. Es funktioniert genau so, wie sie es erwarten. Neue AbiWord Benuzter werden also nicht verwirrt.
