Einfügen von Objekten

Um die Schaffung Ihres Dokumentes zu vervollständigen, erlaubt AbiWord es Ihnen, verschiedene Objekte in Ihr Dokument einzufügen.

   
  • Datum und Zeit
  • Spezielle Zeichen
  • Bilder
  • Felder


Datum und Zeit einfügen

Um automatisch das aktuelle Datum und die aktuelle Zeiz einzufügen, poisitionieren Sie den Cursor an die Stelle, an der sie diese einfügen wollen. Klicken Sie im "Insert" Menu auf "Date and Time".

Ein Fenster mit verschiedenen Formaten für das Datum und die Zeit erscheinen. Wählen Sie das gewünschte Format aus und klicken Sie auf "OK", um diese einzufügen oder auf "Cancel" um dies abzubrechen cancel.

Symbole

Spezielle Symbole und Zeichen, die sich nicht auf der Tastatur befinden, werden eingefügt, indem Sie den Cursor an die Stelle, an der Sie diese einfügen wollen, positionieren und dann eine der folgende Aktionen ausführen:

  1. Im "Insert" Menu das Symbol "Symbol..." anklicken.

    • Das folgende Fenster wird erscheinen:

    • Wählen Sie den Schriftsatz aus indem Sie auf den Pfeil am obersten Feld drücken.
    • Klicken Sie auf das gewünschte Zeichen auf dem Gitte.
    • Ist das Zeichen vergrößert unten sichtbar, klicken Sie "Insert".
    • Wiederholen Sie dieses so oft, wie sie es brauchen.
    • An Ende klicken Sie den "Close" Knopf.

  2. Verwenden Sie das Tastaturkürzel "Ctrl " und "M".


Bilder

AbiWord kann Bilder in Ihr Dokument einfügen. Folgende Bildformate werden unterstützt:
  • *.png (Portable Network Graphics) Format
  • *.bmp (Windows Bitmap) Format
  • *.svg (Scalable Vector Graphics) Format

Um ein Bild einzufügen, positionieren Sie den Cursor an die Stelle, an der das Bild erscheinen soll und klicken Sie im "Image menu" auf "Image...".

  • Folgendes Fenster wird erscheinen:

     

Geben Sie den Pfad, unter dem Ihr Bild sich befindet, oder browsen Sie bis zu Ihrem Bild. Klicken Sie den "Open " Knopf zum Einfügen des Bildes.

AbiWord stellt das eingefügte Bild als ein Zeichen dar. So ist es ein Objekt, das an Ausrichtungen und Linienabstände gebunden ist und mit der Leertaste oder Tabulatortaste bewegt werden kann.

Felder

Um ein Feld in Ihr Dokument einzufügen, klicken Sie im "Insert" Menu auf "Field". Dies wird ein Feld-Dialog hervorbringen. Folgende Dinge können Sie auswählen:

  • aktuelles Datum und Zeit
  • Seitennummer
  • Anzahl der Seiten
  • Feldname
